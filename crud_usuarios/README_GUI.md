# GUI - CRUD de Usuários

Esta pasta contém uma versão com interface gráfica (Tkinter) do CRUD de usuários.

## Arquivo principal
- `gui.py` — Interface gráfica com cadastro, listagem, atualização e remoção.

## Como executar
```bash
python3 gui.py
```

Observação: o banco `usuarios.db` será criado na primeira execução (mesmo banco usado pela versão CLI).
